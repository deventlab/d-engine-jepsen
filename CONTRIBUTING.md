# Contributing to d-engine-jepsen

## Reporting Issues

Include test output, linearizability report, d-engine version, and test config.

## Pull Requests

1. Ensure `make test` passes
2. Follow existing code structure in `src/jepsen/d_engine/`
3. Update README if adding features
4. Use commit format: `<type> #<ticket>: <description>`

Types: `feat`, `fix`, `docs`, `test`, `refactor`

## Questions?

Open a GitHub Issue or Discussion.
